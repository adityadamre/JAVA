# JAVA — Data Structures & Algorithms in Java

## Overview

This repository contains implementations of core data structures and algorithms in Java, along with solved practice problems. It serves as a personal learning log and reference collection built through consistent practice of Data Structures & Algorithms (DSA). It is designed to help build a strong foundation in algorithmic programming using Java.

## Topics Covered

### Data Structures

- **[Arrays](Arrays/)** / **[TwoD_Arrays](TwoD_Arrays/)** / **[ArrayList](ArrayList/)**: Static and dynamic arrays, multi-dimensional structures.
- **[Strings](Strings/)**: String manipulation, searching, and pattern matching.
- **[LinkedList](LinkedList/)**: Singly, doubly, and circular linked lists.
- **[Stack](Stack/)** & **[Queue](Queue/)**: LIFO and FIFO structures, monotonic stacks/queues.
- **[Hashing](Hashing/)**: Hash maps, hash sets, and frequency counting.
- **[Trees](BinaryTrees/)** / **[BinarySearchTrees](BinarySearchTrees/)**: Tree traversals, BST properties, and structural operations.
- **[Heaps](Heaps/)**: Priority queues, min-heaps, and max-heaps.
- **[Graphs](Graphs/)**: Directed/undirected graphs, BFS, DFS, shortest paths, and spanning trees.
- **[Tries](Tries/)**: Prefix trees for string-related algorithms.
- **[SegmentTrees](SegmentTrees/)**: Range query structures and lazy propagation.

### Algorithmic Paradigms & Techniques

- **[Basics](Basics/)** / **[Math](Math/)**: Fundamental programming constructs and mathematical concepts.
- **[BinarySearch](BinarySearch/)**: Search space reduction and monotonic function optimization.
- **[TwoPntr_SlidingWindow](TwoPntr_SlidingWindow/)**: Linear time optimization strategies for contiguous sequences.
- **[Bit_Manipulation](Bit_Manipulation/)**: Bitwise operations for space and time efficiency.
- **[Greedy](Greedy/)**: Local optimum choices for global optimization.
- **[Backtracking](Backtracking/)**: Exhaustive search with pruning (combinations, permutations, pathfinding).
- **[DynamicProgramming](DynamicProgramming/)**: State transitions, memoization, and tabulation techniques.
